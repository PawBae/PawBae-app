# Skin template · 皮肤模板

1. 画一张 1536×1872 的精灵图集（192×208 格子，8 列 × 9 行，一行一个动画），存成 `spritesheet.webp` 放进本文件夹。行布局见 [SKIN-SPEC.zh.md](../SKIN-SPEC.zh.md)。
2. 改 `pet.json` 里的 `id` / `displayName` / `description`。
3. 在 PawBae 里打开 皮肤工坊 🎨 → 导入文件夹，选择本文件夹。

---

1. Draw a 1536×1872 spritesheet (192×208 cells, 8 cols × 9 rows, one animation per row), save it as `spritesheet.webp` in this folder. Row layout: [SKIN-SPEC.en.md](../SKIN-SPEC.en.md).
2. Edit `id` / `displayName` / `description` in `pet.json`.
3. In PawBae, open Skin Workshop 🎨 → Import folder and pick this folder.
