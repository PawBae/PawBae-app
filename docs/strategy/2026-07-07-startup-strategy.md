# PawBae 创业战略 v1(2026-07-07)

> 基于 12 个研究 agent 的市场调研(桌宠竞品 / AI 陪伴产品 / agent 采用鸿沟 / 开发者细分市场 / 变现模式)+ 代码库摸底 + 四视角功能构思 + 一轮对抗性批判综合而成。数据点来自公开来源(Steam、Pew、Menlo Ventures、公司公告与媒体报道),截至调研当日。

## 1. 一句话定位与时机

**定位:PawBae 不是"agent 监控器",而是"你的 AI agent 的脸"——一只有性格、有记忆、会成长的桌面宠物,把看不见的 agent 变成看得见、摸得着、可信任的伙伴。**

为什么是现在:

- **采用鸿沟真实且巨大**:约一半美国成年人已在用 AI 聊天机器人(Pew 2026),但 agent 使用仍是极小切片——全球约 18 亿 AI 用户中只有约 3% 付费(Menlo)。阻碍不是技术而是**情绪**:39% 的观望者中 80% 更信任人、53% 需要"有人负责"。宠物恰好正面攻击这两点:它给 agent 一张脸、一个可见的工作状态、一种"有人在看着"的问责感。
- **窗口已开但未被占据**:巨头把 agent 包装成浏览器(Atlas)、终端(Claude Code)或企业工具($200/月的 Operator)。微软 Mico 验证了"化身"方向但只是语音模式附件。没有人在做"友好的常驻桌面伙伴"。
- **滩头市场正在爆炸**:Claude Code 半年做到 $10 亿年化、Codex 500 万周活(半年 8 倍)、Cursor $40 亿 ARR。这条铁轨自己会通向大众:Codex 用户 20% 已是非开发者,Claude Cowork 用量 90% 与写代码无关。
- **威胁同样是佐证**:Clawd on Desk 四个月 5.2k star、Anthropic 内部有 Claude Buddy——赛道已被验证,拼的是速度和深度。

## 2. 目标用户分层

开发者不是终点,是**分发渠道**(Figma / Lovable 式自下而上)。

| 阶段 | 用户 | 依据与门槛 |
|------|------|-----------|
| 滩头(现在–6 月) | Agent 重度开发者 | CC/Cursor/Codex 合计数百万周活且年翻倍;已付 $20–200/月订阅;分发几乎免费。门槛:约 1 万周活、D30 留存 >20% |
| 扩展(6–18 月) | 顺 agent 铁轨来的非技术用户 | Codex 周活的 20%(约百万人)已是非开发者;需求是 Cowork 验证过的浅任务(整理下载/收据转表/总结 PDF;Menlo 前十:邮件 19%、研究 18%、报销 15%) |
| 远期(18 月+) | 主流消费者 + B2B | 消费端对标 Finch($30–40M ARR 零融资);B2B 是"团队吉祥物 + AI 采用度游戏化"(HeyTaco $3/人/月 × 3000+ 团队) |

## 3. 核心游戏循环:「照护倒置」——干活就是养宠

Finch 用"你做真实的自我关怀 → 小鸟成长"零融资做到 $30–40M ARR;中国陪伴 App 用"聊天时长"做循环则 D3 留存 <20% 且推理成本压死定价。结论:**宠物的成长必须绑定真实完成的 agent 工作,而不是聊天量。**

一个闭环,五个节拍:

1. **委托**——你启动 agent 任务(未来:把文件拖给宠物);
2. **陪伴**——宠物实时映射工作状态,长任务时"背包去冒险"(Travel Frog 的 3000 万下载证明"期待感"比"义务感"强);
3. **收获**——任务完成 = 宠物开饭(token 换算成食物)+ 带回随机纪念品;
4. **成长**——食物与金币驱动五阶段进化和三性格分支,宠物日记记住你们一起干过的活——**记忆成为换用成本**;
5. **展示**——周报卡片、图鉴、皮肤,晒出去的每一张卡都是获客素材,回到第 1 步。

**设计红线**:永不惩罚(饿了只打盹,不死不哭穷——Finch 铁律)、永不打扰(Clippy 之死、Mico 之生的教训)、远离恋爱/NSFW/未成年人(Replika、Character.AI、Ani 的所有重大翻车都源于此)。

## 4. 三阶段路线图

工作量标注:S=天级,M=周级,L=月级。

### 阶段一(0–3 月):把循环焊死,把分发修通
目标:核心循环完整可玩,D30 >20%,进入 Steam 视野。

- Token 喂养循环 [S] —— **已实现**,见 feat/token-feeding-loop 分支与 `docs/superpowers/specs/2026-07-07-token-feeding-loop-design.md`
- 叼来审批单 [S] —— agent 等审批时宠物叼纸条来找你,点击跳转终端,快速响应得好感度
- 每日任务板 + 宽容连胜 [S]
- Agent 冒险 + 纪念品架 [M]
- 每周 Paw 报告分享卡 [S](viberank 证明开发者爱晒 agent 账单)
- 代码签名 + Steam 页面攒愿望单 —— 不是功能,是一切增长的前提

### 阶段二(3–9 月):留存加深 + 社区 + 首次营收
目标:UGC 飞轮转起来,中国市场点火,验证 2–5% 付费转化。

- 皮肤工坊 UGC [M] —— VPet 靠 745+ 模组换来 5 万条 98% 好评;Desktop Mate 禁模组被刷到 60% 差评。最强留存引擎 + 中国楔子,必须抢在克隆前
- 孵蛋与物种图鉴 [M](Claude Buddy 有 18 物种,正面防御平台内置竞争)
- 首批付费皮肤包 $3.99–7.99 上架 Steam [M]
- 宠物日记 + 早安问候 [M] —— 模板生成、零推理成本的"记忆",最强依恋杠杆
- 小队与排行榜 [M](AgentPet 已抄走排行榜概念,这是抢跑战)
- 直播模式 OBS 透明源 [S](Neuro-sama 单靠人格月入 $40 万+,vibe coding 主播就是目标用户)

### 阶段三(9–18 月):宠物成为界面(大众化)
目标:非技术用户 10 分钟内完成第一次安全委托,验证消费订阅。

- 投喂任务 Drop-to-Do [M] —— 拖文件给宠物,它用人话提议一个动作;首发三个动词:整理下载、收据转表、总结 PDF
- 爪印日志 + 大红爪(玻璃箱信任层)[M] —— 操作气泡直播、一键撤销、常驻暂停、干跑预览
- 任务书:头七个小任务 [S]
- 宠物大脑分层 [L] —— BYOK 免费 / $6–10 月托管 / 本地小模型
- 领养计划 [M] —— 开发者给父母/伴侣配宠物,用已建好的 OpenClaw/SSH 远程设施零账号上手

## 5. 商业模式:三层火箭,核心永久免费

1. **免费核心 + 化妆品**(3–9 月启动):Steam 皮肤包 $3.99–7.99,按 2–5% 转化规划,化妆品占收入约 80%;联名 IP $9.99–14.99 留作鲸鱼产品。
2. **PawBae Pro $8/月或 $59/年**(第 6 月起):跨 agent 统计、token 开销分析、SSH/远程舰队监控、多宠、BYOK 语音。客群已在为 agent 付 $20–200/月。
3. **消费订阅 $9.99/月**(12 月后,Finch 模板)+ **B2B 试点 $3–5/人/月**。18 个月时砍掉转化差的那条。

**明确拒绝**:打包 token 的固定订阅(Cursor 改价翻车、星野 ¥12/月盖不住推理成本)、单角色付费墙 + 禁模组(Desktop Mate 60% 差评)、纯靠病毒流量(Bongo Cat 19 万在线月赚 $3k)、LLM 聊天作为核心付费点(成本 + 监管翻车源头)。

## 6. 增长与分发(顺序执行)

1. 先签名公证(Apple $99/年 + Windows Azure Trusted Signing)——"未知开发者"警告毒化一切渠道;
2. GitHub/X 开发者病毒层(此赛道免费工具几个月 5k–27k star);
3. Steam 愿望单战役(Wallpaper Engine 约 3500 万拥有者、Rusty's Retirement 55 万份证明桌面玩具品类);
4. 中国第二引擎:中文语音 + zh 本地化是现成武器;VPet 4.27 万条中文好评;bilibili 开发日志 + VPet 社区 + 小红书/即刻;
5. 产品即广告:直播模式 + 领养计划。Product Hunt 只当一日烟花。

## 7. 护城河与竞争风险(诚实版)

**今天没有结构性护城河**——AgentPet 几周内抄完全部规格;Anthropic 有 Claude Buddy。现在的优势只有速度与工程质量。按序构建三条真护城河:

1. **角色 IP**(有名字、lore、人格——换掉宠物 = 抛弃朋友);
2. **工作记忆**(几个月的共同工作史抄不走);
3. **UGC 网络效应**(Workshop 生态 fork 不走)。

| 风险 | 证据 | 预案 |
|------|------|------|
| 平台自建宠物 | Claude Buddy 已存在 | 做多 agent 的"瑞士" + 角色 IP |
| OS 级化身 | Mico 已进 40 市场 | 跨平台 + 深一个数量级,迭代快过平台 |
| 新鲜感流失 | Travel Frog 衰减;陪伴 App D3<20% | 照护倒置 + 宽容连胜 + Workshop 内容流 |
| 克隆速度 | AgentPet 几周抄完规格 | 抢发社交/身份循环,记忆即换用成本 |
| Token 成本 | Cursor 翻车;星野死亡螺旋 | 只做 BYOK 或硬封顶 |

## 8. 最近 30 天行动清单

1. 第 1 周:签名(Apple Developer $99 + Azure Trusted Signing)——最高杠杆单项;
2. 第 1 周(并行):**匿名 opt-in 遥测**(安装、D1/D7/D30、日均 agent 会话)——没有它一切门槛都是盲猜;
3. 第 1–2 周:Token 喂养循环(已完成,待合并);
4. 第 2–3 周:叼来审批单 + 每日任务板;
5. 第 3–4 周:周报分享卡(canvas 导出 PNG,en+zh);
6. 第 4 周:Steam 开发者账号 + 上架页面($100),攒愿望单;
7. 给角色起名、写 200 字 lore——护城河第一块砖;
8. 发布战役:X 长帖 + bilibili 视频 + 即刻/V2EX/小红书 + 10 个 vibe coding 主播 DM;
9. 访谈 20 个用户:"哪个瞬间你笑了" + "哪天你没打开它、为什么";
10. 公开阶段门槛:3 个月 1 万安装、D30>20%、每周 3+ 自发晒卡——不达标先修循环再谈扩张。

## 9. 顾问批注:六个必须直面的盲点(对抗性审查)

1. **核心命题未验证**——"卡通脸 = 敢授权 AI"是逻辑跳跃。两周内做 15 人假门测试(宠物提议整理下载文件夹,数多少人点确认),低于一半就重写阶段三。
2. **开发者圈已证明的恰恰是"没人付钱"**(ccusage 17k 星零收入;1 万周活 × 3% × $5 ≈ $1,500/月)。现在就挂预售页测付费意愿,别等第 6 个月。
3. **对 Clawd on Desk 的轻描淡写是自我安慰**(5.2k 星、三平台、14 agent、中文媒体已报道)。访谈 10 个它的用户,验证差异化叙事是否成立。
4. **遥测顺序错误**——所有阶段门槛都依赖数据,但至今零遥测。挪到第 1 周(已在上方行动清单修正)。
5. **阶段三是第二次创业,不是"扩展"**——误操作责任、托管成本、非 GitHub/Steam 的获客渠道。先做穷人版领养测试(5 个用户给父母装),计划里写明"何时决定不做它"。
6. **单人带宽 vs 三线作战**——这是 5 人团队的计划。把 30 天清单砍到 5 项,写出公开的"明确不做"清单;写不出来说明还没做选择。
